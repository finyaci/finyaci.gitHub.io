sscanf()拆包 sprintf()组包

  • sscanf() sprintf() 从字符串中获取输入，格式化输出到指定数组字符串
  • sscanf() 字符串默认以空格分割，使用其他分割会产生错误

strchr() strstr() 字符、字符串查找

  • strchr()匹配所给的字符，返回一个指向char类型的地址值
  • strstr()匹配所给字符串，返回地址存放在指针中

strtok()字符串切割

  • 切割成功返回地址，失败返回NULL
  • 第一次调用写要切割的字符串
  • 第二次调用写NULL
  • 原先字符串会被破坏

11 指针

%p通常以十六进制显示指针的值

多级指针

  • 取值操作 *

*p <==> *(p+0) <==> p[0] 指针加减法

  • 指针加减法要根据指针的类型变换

万能指针

  • 万能指针在使用（存放数据而不是地址值）时要强制类型转化，否则编译器不知道操作的长度

const 修饰指针

  • 修饰指针变量： 地址不能修改
  • 修饰* ： 指针所指的变量内容只读
  • const int p1; //const 修饰 ，指的是指针变量所指的内存为只读，但指针变量可修改
  • int const * p2; //等价写法
  • int * const p3; //const 修饰指针变量，指针变量为只读不可修改，但指针变量所指的内存可修改
  • int const * const p4; // 指针变量为只读，其所指向的内存也为只读
  • 靠的近的只读

指针操作数组

  • *(p+i)

二维数组指针操作

  • *(ary[n]+i);
  • *(*(ary+n)+i);
  • ary[n][i];

二维数组补充

  • int ary[2][3]
  • ary 是一个二级指针，**ary 的值为 ary[0][0]
  • *ary = ary[0] = &ary[0] = &ary[0][0]
  • *(ary+1) = ary[1] = &ary[1] = &ary[1][0]
  • *ary[1] = ary[1][0] = **(ary+1)

数组指针(二维数组较多)

  • int ary[ROW][COL];
  • int (*p)[COL] = ary;
