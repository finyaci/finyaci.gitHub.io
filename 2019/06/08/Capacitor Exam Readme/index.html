Capacitor Exam Readme

Posted on 2019-06-08 Edited on 2019-07-02

程序运行顺序，

Image_train.py -> capcitor_exam.py -> capacitor_exam_auto.py

程序运行结果在result.txt中

Image_train.py

程序简要说明

该程序主要用于训练卷积神经网络网络，电容数据存放在’./Database/‘文件夹下，通过读入’./Database/‘文件夹下电容图像，对网络进行训练，训练的参数文件存放在’./model/‘文件夹中，之后即可使用该参数文件编写程序对图像进行预测。

读取图像并处理read_img()函数说明

def read_img(path)

形参说明：
path - 训练图像的路径

该函数读取path目录下所有子文件夹下的电容图像，原电容图像大小是8181，将其全部调整大小为100100。朝左的图像标记为0，朝下标记为1，朝右标记为2，朝上标记为3，空标记为4.具体情况运行程序时会显示。
调整后的图像存放在imgs的列表中，标签存放在labels列表中。该函数最后会将imgs列表和labels列表转换成np的array格式并返回，便于数据的使用。

划分训练集和验证集

划分数据集前将数据打乱
num_example存放图像的总数目
np.arange(num_example)生成num_example个索引存放在arr中，打乱这些索引，将索引重新赋值给data和label实现了数据的打乱

x_train、y_train 分别为训练集的图像和标签
x_val、y_val 分别为验证集的图像和标签

其中80%为训练集，20%为验证集

capcitor_exam.py

程序简要说明

该程序主要人工获取电路板上电容位置坐标、电容大小和正确电容负极朝向等各项参数的程序。其中各项参数会被保存为coord.pk文件，方便capacitor_exam_auto.py程序进行调用。

变量定义说明

flower_dict = {0: ‘left’, 1: ‘down’, 2: ‘right’, 3: ‘up’, 4: ‘empty’}
电容印花定义，与Image_train.py中定义要一致
x_length, y_length - 以光标为中心x方向偏移，y方向偏移
xy_list - 坐标
length_list - 偏移
dir_list - 预测与实际结果是否一致性存放列表
idx - 截取的元件序号，设置为1

capture()函数说明

def capture(event, x, y, flags, params) 对图像截取

配合OpenCV的鼠标事件使用：
cv2.setMouseCallback(‘img’, capture)

全局变量说明：

cap_img - 被截取的图像
start_flag - 等待标志，防止出错
x_length、y_length - x、y方向上的偏移量
xy - 临时变量，元组类型，保存(x, y)坐标，xy将会被添加至xy_list中。

img - 测试图像，原图
img_bk_for_rect - 备份图像，用来绘制截取框
img_bk_for_cap - 备份图像，用来截取图像

不在原图上绘制截取框，防止图像截取时，元件靠的过近，一旦绘制了截取框，将会把靠的过近截取框也截下来，影响网络的判断。

主要功能说明：

鼠标中键（滚轮）按下 按下一次，x_length、y_length各减少10，截取框减小，并会在图像上显示出蓝色的方框

鼠标右键按下 按下一次，x_length、y_length各增加10，截取框增大，并会在图像上显示出绿色的方框

鼠标滚轮滚动 实现对图像重新载入，以实现清除的目的

鼠标左键双击 根据x_length、y_length大小在图上显示白色的截取框，右上角显示方框选取的次序

预测部分说明

预测截取的cap_img图像时，首先要安照常规定义TensorFlow的方式建立会话，并读入参数文件，代码如下

with tf.Session() as sess:
saver = tf.train.import_meta_graph(‘./model/model.ckpt.meta’)
saver.restore(sess, tf.train.latest_checkpoint(‘./model/‘))

对图像进行预测时同样要调整截取图像的大小
r_cap_img = cv2.resize(cap_img, (100, 100), interpolation=cv2.INTER_LINEAR)

调用先前网络的参数对截取的图像进行预测，将打印出预测矩阵和当前样本预测的方向。如果预测正确，输入’y’则会记录当前预测的方向，结果将会加入dir_list列表；如果预测错误，应当输入’n’，将会删除元件坐标等数据。

保存参数

with open(‘coord.pk’, ‘wb’) as f:
pickle.dump(xy_list, f)
pickle.dump(length_list, f)
pickle.dump(dir_list, f)

在当前目录下，会以二进制的形式保存坐标、偏移量和正确朝向的参数。参数保存在coord.pk文件下。用于capacitor_exam_auto.py程序调用

capacitor_exam_auto.py

程序简要说明

capacitor_exam_auto.py即是对电路板图像上电容插件方向检测的程序。该程序首先会载入coord.pk文件，获取待检测的图像和位置坐标等信息。对电路板上电容部件检测的结果将会存放在当前目录的result.txt文件中，其中图像预测部分循环和capacitor_exam.py没有太大区别。

变量定义说明

img_path - 存放待检测的图像路径，根据图像的名称和路径，可以自行定义
predicted_res_list - 网络预测结果
res_list - 网络预测结果和正确朝向比对的判断结果
idx_list - 存放电容序号
x, y - 坐标
x_length, y_length、xy_list、 length_list、 dir_list与capacitor_exam.py文件中定义一致

capimgbyxy()函数说明

def capimgbyxy()

函数说明：
根据x, y 、x_length, y_length、对图像自动截取

exam()函数说明

def exam(predicted_flag)

形参说明：
predicted_flag - 等待标志，防止出错

函数说明：
将预测的结果和应当正确的结果进行比对，如果朝向一致，则在截取框上用绿色显示正确朝向，如果错误，则显示红色的’wrong’字样。同时，比对结果将被加入res_list中

for coord in xy_list:

图像预测部分循环和capacitor_exam.py没有太大区别。
