Linux Basic

Posted on 2019-01-24 Edited on 2019-05-29

1 目录简介

  • /usr/bin 可执行二进制文件
  • /lib 库文件
  • /usr/include 头文件

2 文件类型

  • d 目录文件
  • - 普通文件
  • l 链接文件 link
  • c 硬件字符设备 char /dev
  • b 硬件块设备 block
  • s 管道文件

3 文件权限

-rw-r—r— root root 文件所有者 用户组 其他用户 文件所有者 文件属于用户组
r read
w write
x exe

  • whoami
  • cat /etc/group
  • cat /etc/passwd

4 命令查询

  • command [-options][parameter1]

—help 查询

  • ls —help 查询ls的用法

man 查询

f 前滚
b 后滚
h 所有功能键

  • enter 滚一行
  • space 下一屏

  • /word 搜索word

  • man man

  • man 1 printf
  • man 3 printf

查询章节

  • whatis printf

  • output:

    printf (1) - format and print data
    printf (3) - formatted output conversion

which 查看命令的存放位置

  • which gcc
  • which ls
  • which rm

5 文件管理

ls & 输出重定向

  • ls 1.txt 可用于搜索
  • ls -lah > 2.txt 输出重定向

  • ls >> 2.txt 追加

  • cat 1.txt 2.txt > abc.txt 将12的内容取出重定向输出到abc

分屏显示

  • more 2.txt

| 管道

一个命令的输出可以作为另一个命令的输入

  • cd /usr/include/
  • ls | grep std

output:

stdc-predef.h
stdint.h
stdio_ext.h
stdio.h
stdlib.h
unistd.h

  • ls /usr/include/ | more
    分屏显示

清屏

  • ctrl+l
  • clear

获取当前路径

  • pwd

mkdir rmdir rm

  • mkdir a/b/c -p 递归创建
  • rmdir 无法递归删除
  • rm a -rf 递归删除文件夹且不提示
  • rm * -rf 慎用，全部删除

ln 建立链接文件

链接普通文件

  • ln -s 源文件 链接文件

    链接目录

  • ln -s 源目录 链接目录

grep 文本查找

  • grep mike text.txt -n 显示行号
  • grep mike text.txt -i 忽略大小写

  • grep mike text.txt -v 取反

  • grep ^a text.txt -n 查找以a开头的行

  • grep ke$ text.txt -n 查找以ke结尾的行
  • grep 1 2 3 text.txt 查找关键字1或2或3
  • grep “1 2 3” text.txt 查找1 2 3整体

find 查找文件

  • find ./ -name *.c
  • find /usr/include/ -name stdio*
  • find ./ -size +12k -size -16k
  • find ./ -perm 0777 搜索权限777的文件

cp 拷贝

  • cp src des -r 文件夹拷贝
  • cp src.c des.c 文件拷贝
  • -f 无提示 -v显示拷贝进度
  • cp /lib ~/Test -rv 将/下lib文件夹拷贝至home的Test文件夹下

mv

  • mv mike.c aaa.c 改名
  • mv mike.c ~/Test/bbb.c 移动到Test下并改名

file 查看文件类型

  • file \

tar 打包

  • tar -cvf name.tar * 当前目录下所有文件打包
  • tar -cvf name.tar 1.txt 2.txt
  • tar -xvf name.tar 解包
  • tar -xvf name.tar -C ~/Test/1 解包到指定目录要加-C

查看文件夹大小

  • du ~/Test -h

gzip 压缩

  • gzip [-r] xxx.tar 压缩成tar.gz
  • gzip -d xxx.tar.gz 解压成tar文件

压缩打包

  • tar -cvzf name.tar.gz 1.txt
  • tar -xvzf name.tar.gz

bzip2

  • bzip2 [-z] xxx.tar
  • bzip2 -d xxx.tar.bz2

压缩打包

  • tar -cvjf name.tar.gz 1.txt
  • tar -xvjf name.tar.gz

zip

  • zip压缩文件的目标文件不需要指定扩展名
  • zip -r(必须要加此参数) 目标文件(没有扩展名) 源文件
  • zip -r test *

unzip

  • unzip -d 路径(必须写) 解压文件
  • unzip -d ./ test.zip

6 用户管理

who whoami w

  • whoami
  • who
  • w

su 用户切换

  • sudo su [root] 切换到root用户
  • sudo su - [root] 切换root用户并改变当前路径为root的目录
  • sudo su finyaci 切换到普通用户
  • sudo su - finyaci 切换到普通用户并改变当前路径

groupadd groupdel 用户组增删

  • cat /etc/group 查看 用户组名 用户组id
  • sudo groupadd g1
  • sudo groupdel

usermod useradd userdel 用户增删改变用户组

  • cat /etc/passwd 查看用户信息 用户名 用户id 用户组id
  • sudo useradd -d /home/finyaci finyaci -m -d 指定主目录 用户名 -m如果指定目录不存在自动创建目录
  • sudo useradd -d /home/g2u1 g2u1 -m -g g2 -g 指定用户组
  • sudo usermod -g \ \ 改变用户所属的组别
  • sudo uerdel \ 删除用户
  • sudo uerdel -r \ 删除用户并删除主目录

更改密码 passwd

  • sudo passwd g2u1

查看登录记录

  • sudo last

chmod 修改文件权限

chmod u/g/o/a +/-/= rwx 文件
r—:4 -w-:2 —x:1 —-:0

  • chmod u-w 1.txt
  • chmod u=rx 1.txt
  • chmod 777 1.txt

> 对于文件夹:
> 没有r权限则无法查看文件夹内容
> 没有x权限则无法进入文件夹
> 没有w权限则无法在文件夹内新建或删除内容

chown chgrp 修改文件所有者 修改文件所属组

-rw-r—r— root root
文件所有者权限 用户组权限 其他用户权限 文件所有者 文件属于用户组
chown 用户名 文件或目录

  • sudo chown g2u1 2.txt
  • sudo chgrp g2 2.txt

6 系统管理

cal date 查看日历 查看当前日期

  • cal
  • date

ps 查看进程信息

-a 显示终端上所有进程
-u 显示进程详细状态
-x 显示没有终端控制的程序
-w 加宽显示
-r 显示正在运行的程序

  • ps -aux

借助管道过滤进程信息

  • ps -aux | grep g2u1

kill

  • kill \ 终止进程
  • kill -9 \ 杀死进程

top 动态显示进程

M：内存排序
P：CPU排序
T：运行时间排序
U：用户名筛选进程
K：输入PID杀死进程
q 退出 h 帮助

  • top
  • top -d \ 指定更新时间间隔

ctrl+z 切换到后台

& jobs fg 后台程序

& 在后台运行

  • vi hello.c set showcmd
    set number
    ```

基础操作

  • vi hello.c +5 光标默认第五行
  • vi hello.c 使用vi创建文件并编辑

基础命令&快捷键

退出

  • :wq shift+zz 退出保存
  • :q! 退出不保存
  • :x 退出保存
  • :X 加密
  • :! 暂时离开并执行命令
  • :w filename 写入保存

插入操作

  • i 光标左边插入模式
  • shift+i 行首插入
  • a 光标右边插入模式
  • shift+a 行末插入
  • x 删除
  • dw 删除光标位置到下个单词
  • db 删除光标位置到前个单词
  • daw : delete the word under the cursor
  • caw : delete the word under the cursor and put you in insert mode
  • o 光标下方开启新行
  • shift+o 光标上方开启新行

复制粘贴撤销重做

  • [n]dd 剪切从当前行开始的n行
  • [n]yy 复制从当前行开始的n行
  • p 粘贴
  • u 撤销
  • . 重做

文本定位

  • gg 文档开始
  • shift+g 文档末尾
  • [n]gg 或 [n]shift+g 指定文档位置

文本查找

  • /字符串 查找相匹配的内容
  • n 下一个
  • shift+n 上一个
