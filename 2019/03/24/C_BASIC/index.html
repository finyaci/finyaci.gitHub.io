编译过程

  • 预处理：宏定义展开、头文件展开、条件编译，删除注释，不检查语法
  • 编译：检查语法，生产汇编文件
  • 汇编：汇编器汇编生成目标文件（二进制文件）
  • 链接：加入各种依赖库，生成可执行文件
  • gcc main.c -E main.i 只预处理
  • gcc main.i -S main.s 预处理+编译
  • gcc main.s -c main.o 预处理+编译+汇编
  • gcc main.o -o main 预处理+编译+汇编+链接

3 字符串相关内容

  • “string A” 双引号相当于一个地址
  • “string” 用strlen()的返回值是6，而sizeof()则是7，内存中存储时，自动在字符串尾加上’\0’

  • “\”string\”” 字符串内部使用引号，需要转义符

  • scanf(“%[\^\n]”,str); //以换行符作为字符串输入的结束

  • fgets()存储换行符，gets()丢弃换行符换为空字符，puts()将空字符转化为换行符再输出，fputs()则按存储的输出，不会添加换行符

  • 读或输出单个字符的都是返回EOF；读字符串的成功都是返回第一个参数的地址，错误或者文件末尾返回NULL；输出字符串成功则返回非负值通常为0，失败返回EOF。

  • int getchar(void) (存入对应的ASCII值)char强制转换为int的形式返回读取的字符，如果到达文件末尾或发生读错误，则返回 EOF

  • int putchar(int char) (输出ASCII对应的字符)该函数以无符号char强制转换为int的形式返回写入的字符，如果发生错误则返回 EOF

  • int putc(int char, FILE *stream) 该函数以无符号char强制转换为int的形式返回写入的字符，如果发生错误则返回 EOF。

  • int getc(FILE *stream) 从指定的流 stream 获取下一个字符（一个无符号字符），并把位置标识符往前移动。如果到达文件末尾或发生读错误，则返回 EOF。

  • char *fgets(char *str, int n, FILE *stream) 从指定的流stream读取一行，并把它存储在str所指向的字符串内。当读取(n-1)个字符时，或者读取到换行符时，或者到达文件末尾时，它会停止，具体视情况而定。如果达到文件末尾或发生错误，返回一个NULL。

  • char *gets(char *str) 从标准输入stdin读取一行，并把它存储在str所指向的字符串中。当读取到换行符时，或者到达文件末尾时，它会停止，具体视情况而定。如果成功，该函数返回 str。如果发生错误或者到达文件末尾时还未读取任何字符，则返回 NULL。

  • int puts(const char *str) 把一个字符串写入到标准输出stdout，直到空字符，但不包括空字符。换行符会被追加到输出中。如果成功，该函数返回一个非负值，如果发生错误则返回EOF。

  • int fputs(const char *str, FILE *stream) 把字符串写入到指定的流stream中，但不包括空字符。该函数返回一个非负值，如果发生错误则返回 EOF。

格式化输出+动态数组

  • %m.nf 默认右对齐；m是总字符数（小数点算1个字符），n是小数点后打印的精度；当字符宽度小于实际个数时自动扩大（仅整数部分，小数部分按精度输出）

  • %+m.nf 输出左端会有正号

  • %m.nf 右对齐输出，总字符个数由m决定

  • %-m.nf 左对齐输出，总字符个数由m决定
