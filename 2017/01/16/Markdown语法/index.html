Posted on 2017-01-16 Edited on 2020-09-18

兼容HTLM

　　
　　Markdown兼容HTML

标题

　　
　　标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

# h1　　一级标题
## h2　　二级标题
### h3　　三级标题
#### h4　　四级标题
##### h5　　五级标题
###### h6　　六级标题

列表

　　
列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，
有序列表则直接在文字前加1. 2. 3.
符号要和文字之间加上一个字符的空格。

无序列表

格式:
* 1
* 2
* 3

效果: 　　

  • 1
  • 2
  • 3

有序列表

格式:
1. 1
2. 2
3. 3

效果:

  1. 1
  2. 2
  3. 3

粗体与斜体

　　
用两个 * 包含一段文本就是粗体的语法，
用一个 * 包含一段文本就是斜体的语法。

　　粗体 **粗体** 　粗体

　　斜体 *斜体* 　　　　斜体
