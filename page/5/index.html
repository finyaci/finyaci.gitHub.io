Shadowsocks Configuration

Posted on 2018-12-09

1. 安装 pip（不同的Linux distribution安装方式略有不同，可自行google）

apt-get install easy_install 
easy_install pip

2. 安装shadoswocks

pip install shadowsocks

3. 在/etc目录下创建shadowsocks.json配置文件

vim /etc/shadowsocks.json

{
“server”:”你的服务器IP，注意不是127.0.0.1哦”,
“port_password”:{
“8381”:”随意设定8381端口使用密码”,
“8382”:”随意设定8382端口使用密码”,
“8383”:”随意设定8383端口使用密码”
},
“timeout”:300,
“method”:”aes-256-cfb”
}

4. 启动服务

ssserver -c /etc/shadowsocks.json -d start

5. 开机自动运行

把以下命令写到 /etc/rc.local

sudo ssserver -c /etc/shadowsocks.json -d start

-55-价值是什么-隐藏在商品里的抽象劳动

使用价值-满足人需要的某种属性-自然属性-共有的属性-构成社会财富的物质内容
价值-人类的劳动体力脑力耗费-社会属性-生产者的社会关系（劳动换劳动）

价值和使用价值是对立统一的
对立：不可兼得（为了获取价值要让出其使用价值，为了获得使用价值要付出价值） 
统一：如果没有使用价值，对它劳动是没有价值的；如果有使用价值，但不劳动那也没有价值
PS：空气泥土就没有价值

交换价值-交换价值是一种使用价值同另一种使用价值相交换的量的比例或关系-表现为使用价值和使用价值交换的比例-实际上决定交换比例的是商品的价值

价值的存在以使用价值的存在为前提，有价值的一定有使用价值
使用价值是价值的物质承担者，也是交换价值的物质承担者，价值寓于使用价值，
商品的价值实体是凝结在商品里的无差别人类劳动，价值是交换价值的基础，交换价值是价值的表现形式

劳动的二重性：劳动既有具体劳动也有抽象劳动
具体劳动：生产一定使用价值的不同形式的具体劳动，有用劳动-劳动的自然属性 ————> 使用价值
抽象劳动：无差别的人类劳动-劳动的社会属性 ————> 价值

-56- 价值如何衡量

社会必要劳动时间-正常情况下，制造某种商品的平均劳动时间-决定商品价值
商品价值与时间成正比，与生产率反比

社会劳动生产率增加-单位商品价值量减少-相同时间生产商品数量增多-商品价值总量不变
个别劳动生产率增加-单位商品价值量不变-相同时间生产商品数量增多-商品价值总量增多

影响劳动生产率的因素：劳动者平均熟练程度，科技水平，生产过程的社会结合，生产资料的规模和效能以及自然条件

商品的价值量是以简单劳动为尺度计量的，
复杂劳动等于自乘或多倍的简单劳动
简单劳动和复杂劳动的配比由市场决定

流通手段：商品交换的媒介-1.必须是现实的货币 2.可以不足值-一两黄金可以不足一两-衍生出了纸币-无价值-纸币没有价值尺度的功能

一：私人劳动是指劳动具有私人性质．因为在私有制的条件下．．生产商品者占有生产资料．生产多少．怎么生产以及如何分配都是他私人的事．

要解决私人劳动和社会劳动的矛盾也就是说私人劳动要被社会所承认和接受．．可想而知就必须通过商品交换．而如果交换不成功也就是制造商品的的具体劳动得不到社会的承认．．这就使得抽象劳动不能实现其价值．．这就成了具体劳动和抽象劳动的矛盾．而劳动二重性有决定商品二因素．有形成了使用价值和价值的矛盾．还值得注意的是如果生产者的私人劳动的个别劳动时间高于社会必要劳动时间．就会让生产者在竞争中处于不利位置．甚至破产．这显然又形成了个别劳动时间和社会必要劳动时间的矛盾．归根结底具体劳动和抽象劳动的矛盾．使用价值和价值的矛盾．个别劳动时间和社会必要劳动的矛盾都根源与私人劳动和社会劳动的矛盾．
二：私人劳动和社会劳动的矛盾又决定着私有制商品经济的产生和发展的全过程．．随着社会分工和私有制的发展．私人劳动和社会劳动的矛盾也随之发展．并且推动了简单商品经济的规模和范围不断扩大．是私人劳动和社会劳动的矛盾的进一步发展为社会化大生产和生产资料资本主义私人占有之间的矛盾．进而推进简单商品经济发展为资本主义商品经济．

资本的技术构成：生产资料和劳动力之间的比例
资本的价值构成：不变资本和可变资本的比例 C：V

资本的有机构成：由技术构成决定并反映技术构成的变化的价值构成 C：V
1.技术构成决定价值构成（生产资料和生产力比例决定对应不变资本和可变资本的比例）
