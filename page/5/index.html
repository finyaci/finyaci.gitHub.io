Shadowsocks Configuration

Posted on 2018-12-09

1. 安装 pip（不同的Linux distribution安装方式略有不同，可自行google）

apt-get install easy_install 
easy_install pip

2. 安装shadoswocks

pip install shadowsocks

3. 在/etc目录下创建shadowsocks.json配置文件

vim /etc/shadowsocks.json

{
“server”:”你的服务器IP，注意不是127.0.0.1哦”,
“port_password”:{
“8381”:”随意设定8381端口使用密码”,
“8382”:”随意设定8382端口使用密码”,
“8383”:”随意设定8383端口使用密码”
},
“timeout”:300,
“method”:”aes-256-cfb”
}

4. 启动服务

ssserver -c /etc/shadowsocks.json -d start

5. 开机自动运行

把以下命令写到 /etc/rc.local

sudo ssserver -c /etc/shadowsocks.json -d start
