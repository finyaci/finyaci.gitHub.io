    • Ref-css_combinators
      • element element div p 选择

      • element>element div > p 选择其父元素是

      • element1~element2 p ~ ul 选择前面有

        元素的每个

    • 属性选择器

      • Ref-css_attribute_selectors
        • [attribute] [target] 选择带有 target 属性的所有元素。
        • [attribute=value] [target=_blank] 选择带有 target=”_blank” 属性的所有元素。
        • [attribute~=value] [title~=flower] 选择带有包含 “flower” 一词的 title 属性的所有元素。 - 注释：上面的例子会匹配以下属性的元素：title=”flower”、title=”summer flower” 以及 title=”flower new”，但不匹配：title=”my-flower” 或 title=”flowers”。
        • [attribute|=value] [lang|=en] 选择带有以 “en” 开头的 lang 属性的所有元素。 - 注释：值必须是完整或单独的单词
        • [attribute^=value] a[href^=”https”] 选择其 href 属性值以 “https” 开头的每个 <a> 元素。
        • [attribute$=value] a[href$=”.pdf”] 选择其 href 属性值以 “.pdf” 结尾的每个 <a> 元素。
        • [attribute*=value] a[href*=”w3school”] 选择其 href 属性值包含子串 “w3school” 的每个 <a> 元素。
