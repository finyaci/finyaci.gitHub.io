0 Junit 单元测试

Ref-JDBC

  • DriverManager 驱动管理对象
    • Ref-DriverManager

      • Class.forName(“com.mysql.jdbc.Driver”); 反射将类加载进内存
      • Driver类中静态代码块中有注册驱动的代码
    • 获取数据库连接
      • static Connection getConnection​(String url, String user, String password)
        • url:连接路径(jdbc:mysql://ip:port/dbname)
  • Connection 数据库连接对象
    • Ref-Connection
    • 获取执行sql的对象Statement
      • Statement createStatement()

    • 事务管理
      • void setAutoCommit​(boolean autoCommit)

      • void rollback()
  • Statement 执行静态sql的对象
    • Ref-Statement
    • The object used for executing a static SQL statement and returning the results it produces.
    • 执行sql语句
      • boolean execute​(String sql)
      • int executeUpdate​(String sql) - DML(update,insert,delete) DDL(create,drop,alter)
      • ResultSet executeQuery​(String sql) - DQL(select)
  • ResultSet 结果集对象
    • Ref-ResultSet
    • 常用方法
      • boolean next() - 游标下移一行
      • Xxx getXxx(String columnLabel) - 通过列的名称获取
      • Xxx getXxx(int columnIndex) - 通过列的索引值获取，索引默认从1开始
  • PreparedStatement (extends Statement)执行预编译sql的对象
    • Ref-PreparedStatement
    • 步骤
      • 使用Connection创建对象时传入sql语句，待设置参数用?代替
      • void setXxx(int parameterIndex, Xxx x) - 设置参数

6 数据库连接池

  1. 概念
    • 存放数据库连接的容器，系统初始化后连接池会申请一些连接对象
    • 当用户访问数据库时，会从连接池获取连接对象，访问完毕后会将连接对象归还给连接池
  2. 优点
    • 节约资源，高效访问
  3. 实现
    • 标准接口Ref-DataSource
      • Connection getConnection() - 获取连接
      • Connection.close() - 归还连接 如果Connection对象来自连接池，则调用close()方法不是关闭连接而是归还连接

2 Servlet

2.1 Basic

  • 创建包 - com.tutorialservlet.web
  • 创建 - ServletDeom类实现Servlet接口重写方法
  • 配置web.xml <web-app>标签下

2.2 Servlet生命周期

  • init() - Servlet被创建时执行 加载资源
    • 默认首次访问Servlet被创建
      • 改变创建时机<servlet>标签下配置
        • <load-on-startup>为负数-首次访问时创建Servlet
        • <load-on-startup>为正数-服务器启动时创建Servlet
    • Servlet是单例的
      • 不要定义Servlet的成员变量，或者不要改变其成员变量的值 线程安全
  • service() - 每一次Servlet被访问时执行
  • destory() - 服务器正常关闭时执行，Servlet被销毁前执行 释放资源

2.3 Servlet注解配置

  • 使用@WebServlet注解
    • 在Servlet接口的实现类上使用该注解
    • @WebServlet(“/servletDemo”) - 默认设置到value中
    • @WebServlet(urlPatterns = {“/servletDemoBAK”,”/servletDemo”}) - 设置多个虚拟路径

2.5 Servlet实现类

Ref-JavaEE7

* GenericServlet(extends from Servlet)
    * To write a generic servlet, you need only override the abstract service method.
* HttpServlet(extends from GenericServlet)
    * Provides an abstract class to be subclassed to create an HTTP servlet suitable for a Web site. A subclass of HttpServlet must override at least one method, usually one of these:
        * doGet, if the servlet supports HTTP GET requests
        * doPost, for HTTP POST requests
        * doPut, for HTTP PUT requests
        * doDelete, for HTTP DELETE requests
        * init and destroy, to manage resources that are held for the life of the servlet
        * getServletInfo, which the servlet uses to provide information about itself

  • HTTP版本

3.1 Request

  • 数据格式
    • 请求行
      • 请求方式 请求url 请求协议/版本
    • 请求头
      1. Host - 请求主机
      2. User-Agent - UA
      3. Referer - 请求源
    • 请求空行
      • 用于分割的空行
    • 请求体
      • GET没有请求体 POST有请求体为请求参数
      • 用于封装POST请求参数

3.1.1 Request请求类型

  • GET请求
    • url有长度限制
    • 表单信息在url中
  • POST请求
    • url无长度限制
    • 表单信息不在url中

3.1.2 Request对象体系

  • Ref-HttpServletRequest

  • 继承关系

    • ServletRequest - javax.servlet
    • HttpServletRequest(extends from ServletRequest) - javax.servlet.http
    • org.apach.catalina.connector.RequestFacade(implements HttpServletRequest)

  • Request功能

    1. 获取请求消息数据

      1. 获取请求行数据
        • 获取请求方式
          • String getMethod()
        • 获取虚拟目录
          • String getContextPath()
        • 获取Servlet路径
          • String getServletPath()
        • 获取请求参数
          • String getQueryString()
        • 获取URI和URL
          • String getRequestURI()
          • StringBuffer getRequestURL()
        • 获取客户机IP
          • String getRemoteAddr()
      2. 获取请求头数据
        • 获取指定请求头名称的请求头值
          • String getHeader(String name)
        • 获取所有请求头名称
          • Enumeration getHeaderNames()
      3. 获取请求体数据
        • 获取字符流
          • BufferedReader getReader()
        • 获取字节流
          • ServletInputStream getInputStream()

    2. 其它功能

      1. 获取请求参数通用方法
        • String getParameter(String name) username=Jack—->Jack
          • Returns the value of a request parameter as a String, or null if the parameter does not exist.
        • String[] getParameterValues(String name) hobby=reading&&hobby=writing—->[reading,writing]
          • Returns an array of String objects containing all of the values the given request parameter has, or null if the parameter does not exist.
        • Enumeration getParameterNames()
          • Returns an Enumeration of String objects containing the names of the parameters contained in this request.
        • Map getParameterMap()
          • Returns a java.util.Map of the parameters of this request.
        • setCharacterEncoding(String charset)
          • 防止中文乱码 req.setCharacterEncoding("utf-8");
      2. 请求转发
        • 服务器内部资源跳转方式
        • 步骤
          1. 调用同request中的方法生成转发器对象
            • RequestDispatcher getRequestDispatcher(String path)
          2. 传递request和response对象进行转发
            • void forward(ServletRequest request, ServletResponse response)
        • 特点
          • 浏览器路径不发生变化
          • 只能转发到当前服务器内部资源
          • 转发是一次请求-浏览器视角下仍然只有一次请求
      3. 共享数据
      • request域：一次请求，用于请求转发的多个资源中共享数据
      • 方法：
        • void setAttribute(String name, Object o)
        • Object getAttribute(String name)
        • void removeAttribute(String name)
      1. 获取ServletContext
        • ServletContext getServletContext()
